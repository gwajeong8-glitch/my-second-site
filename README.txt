### 노블레스 클럽 웹사이트 시뮬레이션 실행 안내

이 프로젝트는 5개 파일로 구성된 프론트엔드 시뮬레이션입니다.
서버 없이도 페이지 전환과 필터링 기능을 경험할 수 있습니다.

1.  파일 준비:
    - index.html
    - style.css
    - script.js
    - data.js
    - README.txt (현재 파일)

2.  이미지 파일 준비 (모두 index.html과 같은 폴더에 있어야 합니다):
    - vip2.jpg (상단 배너)
    - favicon.ico (브라우저 탭 아이콘 - 404 에러 방지용)
    - manager_test.jpg
    - manager_jian.jpg
    - manager_park.jpg
    - manager_kang.jpg
    - manager_hena.jpg
    - manager_yuri.jpg
    - manager_saerom.jpg
    - manager_mina.jpg
    
    *참고: 위 매니저 이미지 파일들은 임시 이름이므로, 고객님의 실제 이미지 파일명으로 변경하거나, 같은 이름으로 임시 이미지를 폴더에 저장해야 화면에 나타납니다.*

3.  실행 방법:
    - 폴더에 모든 파일과 이미지를 넣습니다.
    - 'index.html' 파일을 더블 클릭하여 웹 브라우저로 실행합니다.

4.  주요 동작 테스트:
    - 하단 메뉴 (홈페이지, 매니저 프로필, 이벤트 등)를 클릭하여 페이지 전환을 확인하세요.
    - '서울', '경기도' 등의 지역 태그를 클릭하여 매니저 목록이 필터링되는지 확인하세요.
    - '+ 더 많은 지역' 버튼을 클릭하여 지역 목록이 확장되는지 확인하세요.
